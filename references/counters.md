# Counter system

Timeless AoE2 counter relationships. Cross-check the civ file before recommending a
counter — the civ may lack the unit or its key upgrades. Exact bonus-damage numbers
live in `units.md`.

## Core rock-paper-scissors

| Unit / group | Beaten by | Beats |
|---|---|---|
| Archer line, foot archers | Skirmishers, Mangonels, Knights/Huskarls/Eagles (once on top of them), Shrivamsha | Infantry, Spearmen, slow units, Villagers |
| Skirmishers | Melee anything, Mangonels, Scorpions | Archers, Cavalry Archers, Spearmen (trash war) |
| Knight line | Camels, Pikemen/Halberdiers, Monks, Genoese Xbow/Camel Archer | Archers, Skirmishers, Siege, Villagers, Militia line (roughly even, cost-effective vs most) |
| Camel line | Pikemen, Militia line, Archers/Arbalesters, Mangonels | Knights, other cavalry, Steppe Lancers |
| Spearman line | Archers, Skirmishers, Mangonels, Hand Cannoneers, Militia line | All cavalry (incl. camels, elephants — huge bonus) |
| Militia line | Archers, Hand Cannoneers, Cavalry (kiting), Jaguars/Teutonic Knights | Eagles, Trash units, buildings, Huskarls |
| Eagle line | Militia line, Jaguar Warriors, Teutonic Knights | Archers, Monks, Siege, Skirmishers, Villager raids |
| Cavalry Archers | Skirmishers, Camels+Husbandry, Eagles, Genitours | Slow melee, Spearmen, Monks, foot archers (with micro) |
| Hand Cannoneers | Archers/Arbs (outrange), Onagers, Cavalry | Infantry (massive bonus), Huskarls, Eagles |
| Battle/War Elephants | Monks (convert!), Halberdiers, Camels, Genoese Xbow, mass Arbs | Everything melee, buildings; win by raw stats if unanswered |
| Steppe Lancers | Camels, Halberdiers, Militia line | Archers, trash, raids (attack from 2nd rank = great in mass) |

## Siege

| Unit | Beaten by | Beats |
|---|---|---|
| Mangonel/Onager line | Cavalry, Rams (soak shots), Bombard Cannons (outrange), other Onagers with better micro | Massed foot archers, infantry, Skirmishers, trees (SO cuts forests) |
| Rams | Militia line, Villagers, Onagers to units inside, Mangudai/Cav w/ anti-siege bonus | Buildings, arrow-fire soaking (garrison infantry inside) |
| Scorpions | Cavalry closing distance, Onagers, Bombard Cannons | Massed infantry & archers (pass-through damage), Elephants (bonus) |
| Bombard Cannon | Cavalry snipe, Huskarls | Siege, buildings, slow masses; outranges Onager/Castles |
| Trebuchet | Cavalry/Eagles snipe, opposing Trebs, Bombard Cannons | Castles, buildings, static defenses |

## Monks
- Counter: expensive single units — Knights, Elephants (top-tier vs elephant civs).
- Countered by: Light Cavalry/Hussar (conversion resistance + speed), Eagles, mass archers, own-Monks with Heresy/Faith civs.
- Redemption converts siege/buildings; Atonement converts enemy monks.

## Trash war (no gold left)
Halberdier > Hussar > Skirmisher > Halberdier. Winner is usually the civ with the
best-upgraded trash (check blacksmith gaps + Imperial Skirm/Winged Hussar access) or
a gold-cheap unique unit still flowing from trade.

## Practical counter rules
1. Numbers and upgrades beat theoretical counters. 10 Arbalesters beat 3 Skirmishers.
2. Counters are cost-based: Skirms "beat" archers by trading wood/food vs gold.
3. Mobility is a counter: Knights can simply avoid Pikemen and raid eco instead.
4. Composition beats single-unit counters: Arbs+Halbs, Cav+Skirm, Onager+Halbs cover
   each other's weaknesses. Recommend a pair, not just one unit.
5. Ballistics/Thumb Ring hugely swing ranged fights; Bloodlines/Husbandry swing cavalry
   fights — check the civ's missing-tech list before promising a matchup win.
